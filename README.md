# Marketing Lead Analysis (SQL)
This project demonstrates my ability to analyze **lead quality** and **campaign effectiveness** using SQL, as required for data-driven decision making.

### Key Skills Demonstrated:
* **Data Transformation:** Cleaning raw lead data for analysis.
* **Business Intelligence:** Calculating conversion rates across marketing channels.
* **Documentation:** Maintaining clear schemas for team transparency.

### How to Use:
1. Run `leads_data.csv` in any MySQL environment.
2. Execute `analysis_queries.sql` to generate performance reports.
